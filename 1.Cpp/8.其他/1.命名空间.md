C++11引入了内联命名空间，外层空间可直接访问内联的成员，主要用于库的版本控制。

```cpp
namespace A
{
    inline namespace B
    {
        void call() { std::cout << "B" << std::endl; }
    }
}

int main()
{
    A::call();    // B
    
    return 0;
}
```

C++17引入了嵌套命名空间。

```cpp
namespace A :: B :: C{}
//等价
namespace A{namespace B{namespace C{}}}
```

匿名命名空间内的所有成员只在当前文件可访问，使用匿名命名空间比使用静态成员更合理。

```cpp
// 匿名空间
namespace{}
```
